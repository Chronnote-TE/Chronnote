# Chronnote 产品概览

Chronnote 是一个面向深度知识工作的桌面应用。它不是只用来记录灵感的便签，也不是只用来生成答案的聊天框，而是一个可以长期保存材料、组织问题、追踪思路并反复回到上下文里的知识工作空间。

## 你可以用它做什么

- 写笔记，并让笔记与 PDF、白板、待办产生关联
- 阅读 PDF，保留摘录、批注、结构和后续问题
- 在白板上整理材料、关系、流程和推演过程
- 把待办放回知识上下文里，而不是孤立地管理任务
- 用 AI 基于真实材料提问、总结、分析和继续追问

## Chronnote 关心什么

Chronnote 的核心不是“更快地产生更多内容”，而是帮助你更稳定地完成这个循环：

1. 提出一个真正的问题
2. 找到和问题有关的材料
3. 在材料之间建立关系
4. 得到新的判断或灵感
5. 把判断沉淀回自己的知识系统

## 为什么是本地优先

知识工作常常需要长期积累。Chronnote 希望你的工作空间首先是你自己的：材料、结构、历史和上下文都不应该被一次性的工具体验打散。

本地优先不等于孤岛。它意味着 Chronnote 会优先把可靠性、可控性和长期保存放在前面，再围绕这些基础提供同步、AI 和协作能力。

## 适合谁

- 学生、研究者、写作者、产品经理、工程师
- 需要同时处理长文档、资料库、项目计划和灵感的人
- 习惯把 AI 当作思考助手，而不是把所有判断都交给 AI 的人
