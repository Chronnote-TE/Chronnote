# 常见问题

## 这个仓库是开源仓库吗？

不是。这个仓库是 Chronnote 的公开介绍和文档仓库，不包含 Chronnote 源码。

## Chronnote 有官网吗？

有：[chronnote.top](https://chronnote.top)。

## Chronnote 是本地应用还是网页应用？

Chronnote 当前以桌面端深度知识工作为核心。官网用于介绍、下载和发布信息。

## 可以反馈问题吗？

可以。你可以通过 GitHub Issues、官网联系方式、邮箱或社群反馈问题。请不要上传私人文档或敏感信息。

## AI 会替我判断吗？

Chronnote 的 AI 目标是帮助你阅读、整理、比较和继续追问。最终判断仍然应该由你完成。
