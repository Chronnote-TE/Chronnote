# Chron Engine

Chron Engine 是 Chronnote 中面向深度知识工作的 AI 能力。它的目标不是把所有输入压缩成一句回答，而是让 AI 能围绕真实材料进行结构化阅读、追问和整理。

## 核心方向

- 基于笔记、PDF、白板、待办等材料理解上下文
- 支持轻量回答，也支持更深入的分析与结构化输出
- 尽量保留材料来源、推理线索和后续问题
- 帮助用户继续思考，而不是结束思考

## 典型用法

- 总结一份长文档，并提取可行动的问题
- 对比多份材料中的观点差异
- 根据白板结构生成研究路线
- 把待办和相关资料放在同一个上下文里重新规划
- 从已有笔记中找出遗漏、冲突和下一步方向

## 产品边界

Chron Engine 是辅助思考的工具。它不应该替代用户的判断，也不应该把复杂材料伪装成确定答案。Chronnote 更重视可追问、可回到材料、可长期沉淀的 AI 体验。
